## Summary

### Key Observations

1. **CPU Variance**: CPU results show high variance (std often > 50% of mean), especially for larger graphs
2. **GPU Stability**: GPU results are very stable (std typically < 5% of mean)
3. **GPU OOM Threshold**: GPU fails with OOM when memory allocation exceeds ~4GB (max_alloc_size = 4,294,959,104 bytes)
4. **GPU Speedup**: GPU is typically 5-30x faster than CPU for the same workload

### GPU Memory Limits

- GraphSAGE: OOM starts at 50k nodes × 2.5M edges
- GCN: OOM starts at 50k nodes × 5M edges
- GAT: OOM starts at 50k nodes × 5M edges
